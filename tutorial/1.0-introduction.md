# Welcome to Programming Lightning: Gossip Network & Routing Payments

Welcome to Programming Lightning: Gossip Network & Routing Payments! In this course, we'll dive deep into the Lightning Network's gossip protocol and learn how nodes create authenticated connections with each other and leverage onion routing to privately and securely route payments across the Lightning network. If you're interested in learning how Lightning payment channels work and implementing one from scratch, check out the first module of Programming Lightning: [Intro to Payment Channels](https://replit.com/@austin-f/Programming-Lightning-Intro-to-Payment-Channels).

# Prerequisites

To get the most out of this course, you should have a working understanding of Bitcoin transactions, script, and Lightning payment channels. If you'd like to brush up beforehand, here are a few excellent resources:
- Script (Free): https://learnmeabitcoin.com/technical/script/
- Transactions (Free): https://learnmeabitcoin.com/technical/transaction/
- Transactions + Script ($120): https://www.udemy.com/course/base58-bitcoin-transactions-one/
- Programming Lightning: Intro to Payment Channels: https://replit.com/@austin-f/Programming-Lightning-Intro-to-Payment-Channels
