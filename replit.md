# Lightning Network Routing Tutorial

## Overview
A Rust project for teaching developers how Bitcoin's Lightning Network Routing works.

## Project Structure
- `src/main.rs` - Main Rust application entry point
- `tutorial/` - Folder containing tutorial lessons and exercises
- `Cargo.toml` - Rust project configuration

## Running the Application
Run with `cargo run` to compile and execute the Rust code.
